# Mica v0.1.7

服务器设置简化 + 桌面本地优先。

## 变更
- **服务器设置合并**:原来的「Mica Cloud」和「Self-hosted」两个选项让人分不清——它们其实只差一个地址。现在合成一个 **「远程服务器」**:地址栏默认 `mica.cloudcele.com`,想连自己的服务器改地址即可,旁边有「Use Mica Cloud」一键填回默认。选择就两个:**远程服务器** / **本地(离线)**。
- **桌面默认本地离线**:全新安装的桌面版**直接开写**,不用先注册或联网(像 Obsidian);需要云同步时到设置里切到「远程服务器」登录即可。
  - *已登录 / 已配置服务器的老用户不受影响*,升级后照常连你原来的服务器。
  - Web 端仍默认在线(浏览器没有本地库)。

## 自动更新
本版可从 v0.1.6 自动更新(设置 → About → 检查更新)。
